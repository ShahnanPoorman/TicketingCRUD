# TicketManager

A Java console application for managing support tickets, built as a proof of concept for a future Spring Boot REST API.

## Features
- Create, read, update, and delete (CRUD) tickets
- Input validation with error handling
- Console-based menu interface

## How to Run
1. Clone the repository at a tagged version
2. Open in your preferred Java IDE (IntelliJ, Eclipse, etc.)
3. Run `Main.java`
4. Follow the on-screen menu to manage tickets
